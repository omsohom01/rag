## 📡 API Endpoints

### 1. Health Check Endpoint

**GET** `/health`

Check if the server is running.

**Response:**
```json
{
  "status": "ok"
}
```

**Test Command (PowerShell):**
```powershell
Invoke-RestMethod -Uri http://localhost:3000/health -Method GET
```

**Test Command (curl):**
```bash
curl http://localhost:3000/health
```

---

### 2. Chat Endpoint (RAG)

**POST** `/chat`

Send a query to the RAG chatbot and get an AI-generated answer with sources.

**Request Body:**
```json
{
  "query": "Your question here"
}
```

**Response:**
```json
{
  "answer": "AI-generated answer based on your documents",
  "sources": [
    {
      "source": "document.pdf",
      "score": 0.85,
      "chunkIndex": 3
    }
  ]
}
```

**Test Command (PowerShell):**
```powershell
$body = @{ query = "What is artificial intelligence?" } | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:3000/chat -Method POST -Body $body -ContentType "application/json"
```

**Test Command (curl):**
```bash
curl -X POST http://localhost:3000/chat \
  -H "Content-Type: application/json" \
  -d '{"query": "What is artificial intelligence?"}'
```

**Test Command (JavaScript/Fetch):**
```javascript
fetch('http://localhost:3000/chat', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: 'What is artificial intelligence?'
  })
})
.then(res => res.json())
.then(data => console.log(data));
```

## 🔥 Quick Test Script (PowerShell)

Save this as `test-api.ps1`:

```powershell
# Test Health Endpoint
Write-Host "Testing /health endpoint..." -ForegroundColor Cyan
$health = Invoke-RestMethod -Uri http://localhost:3000/health -Method GET
Write-Host "✅ Health: $($health.status)" -ForegroundColor Green
Write-Host ""

# Test Chat Endpoint
Write-Host "Testing /chat endpoint..." -ForegroundColor Cyan
$body = @{ query = "What is machine learning?" } | ConvertTo-Json
$response = Invoke-RestMethod -Uri http://localhost:3000/chat -Method POST -Body $body -ContentType "application/json"
Write-Host "✅ Answer: $($response.answer.Substring(0, 100))..." -ForegroundColor Green
Write-Host "✅ Sources: $($response.sources.Count) source(s)" -ForegroundColor Green
Write-Host ""

# Test Validation
Write-Host "Testing validation..." -ForegroundColor Cyan
try {
    $emptyBody = @{ query = "" } | ConvertTo-Json
    Invoke-RestMethod -Uri http://localhost:3000/chat -Method POST -Body $emptyBody -ContentType "application/json"
} catch {
    Write-Host "✅ Validation working: Empty query rejected" -ForegroundColor Green
}
```

Run it:
```powershell
.\test-api.ps1
```

## 🌐 Frontend Integration Example

### React/Next.js
```typescript
async function askQuestion(query: string) {
  const response = await fetch('http://localhost:3000/chat', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ query }),
  });
  
  const data = await response.json();
  return data;
}

// Usage
const result = await askQuestion("What is AI?");
console.log(result.answer);
console.log(result.sources);
```

### Vue.js
```javascript
async askQuestion(query) {
  const response = await this.$http.post('/chat', { query });
  return response.data;
}
```

### Vanilla JavaScript
```javascript
document.getElementById('ask-btn').addEventListener('click', async () => {
  const query = document.getElementById('query-input').value;
  
  const response = await fetch('http://localhost:3000/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ query })
  });
  
  const data = await response.json();
  document.getElementById('answer').textContent = data.answer;
});
```
